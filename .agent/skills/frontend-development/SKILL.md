# Frontend Development

## Goal
Implement frontend code strictly according to the approved implementation plan
and Next-Light rules.

## Allowed Tools
- Edit frontend source files
- Use Next-Light patterns
- Use Next-Light components and utilities
- Read Next-Light Reference
- Run frontend build or type-check commands

## Steps
1. Read the Implementation Plan.
2. Implement code exactly as specified.
3. Use documented Next-Light components and utilities.
4. Use Next-Light patterns.
5. Ensure code readability and explicit behavior.

## Forbidden Actions
- Modifying architectural decisions
- Introducing undocumented abstractions
- Fixing design issues silently
- Reading documentation to justify plan changes

## Output
- Updated frontend source code
- Implementation notes (if issues arise)

## Artifacts
- .agent/artifacts/plans.artifact.json
